# 新生入学管理系统 python939

## 项目概述

新生入学管理系统是基于Python语言开发的一套软件，旨在简化和优化高校新生入学的管理流程。系统使用了Django框架，并使用Mysql数据库进行数据存储。

## 技术栈

- **后端语言**: Python
- **开发框架**: Django
- **数据库**: Mysql

## 功能模块

系统主要包括以下功能模块：

- 新生信息管理
- 录取通知管理
- 报到流程管理
- 数据统计与分析

## 系统角色

- 管理员：负责系统维护和高级管理功能操作。
- 教职工：进行新生信息录入、修改和查询。
- 新生：查询个人信息和报到流程。
